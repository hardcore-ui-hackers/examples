# 3D с ховером и нажатием

Взято с акции «Съедобное-несъедобное» у Тинькова. Интересное — цвет тени задается через currentColor:

Розыгрыш
<template>
    <div class="eda-button">
        <div class="eda-button__i">
            <a href="#prizes" class="eda-button__link">
                <div class="eda-button__link-wrapper">
                    <span class="eda-button__link-text">Розыгрыш</span>
                </div>
                <div class="eda-button__shadow"></div>
            </a>
        </div>
    </div>
</template>

<style lang="less">
    .eda-button {
        display: inline-block;

        &__i {
            z-index: 1;
            padding-right: 6px;
            padding-bottom: 6px;
            overflow: hidden;
            transition: color .2s ease-in-out; /* hover effect */
            color: green;

            &:hover {
                color: blue;
            }
        }

        &__link-wrapper {
            background-color: yellow;
            transition: background-color .2s ease-in-out;
        }

        &__link {
            position: relative;
            display: block;
            color: inherit;
            text-decoration: none !important; /* !important due to Vuepress default styles */

            &::after,
            &::before {
                position: absolute;
                z-index: -1;
                content: "";
                width: 0;
                height: 0;
                border-style: solid
            }

            &::before {
                right: -6px;
                top: 0;
                border-width: 6px 0 0 6px;
                border-color: transparent transparent transparent currentColor;
            }

            &::after {
                bottom: -6px;
                left: 0;
                border-width: 0 6px 6px 0;
                border-color: transparent currentColor transparent transparent;
            }

            &:active {
                transform: translate(6px, 6px)
            }
        }

        &__link,
        &__link::after,
        &__link::before {
            transition: transform .2s ease-in-out; /* pressing effect */
        }

        &__link-text {
            font-size: 24px;

        }

        &__shadow {
            position: absolute;
            z-index: -1;
            background-color: currentColor;
            top: 6px;
            left: 6px;
            right: -6px;
            bottom: -6px
        }
    }
</style>

# 3D с ховером и нажатием (2)

Свойство transition-timing-function задаётся с помощью функции cubic-bezier(). Это кубическая кривая Безье, которая строится по четырем точкам и задаёт параметры ускорения анимации.

Click!
Mouse over me!
<template>
    <div class="button-container">
        <a href="javascript:void(0)" class="flipping-button">
            <div class="qube">
                <div class="front">Click!</div>
                <div class="back">Mouse over me!</div>
            </div>
        </a>
    </div>
</template>

<style lang="less">

    //блок позиционирования для вьюпресса:

    .button-container {
        position: relative;
        margin-top: 15px;
        height: 80px;
        overflow: hidden;
    }

    // helper cleasses:
    .centered {
        position: absolute;
        top: 0;
        bottom: 0;
        left: 0;
        right: 0;
    }

    .full {
        width: 100%;
        height: 100%;
    }

    // button style:

    .flipping-button {
        .centered;
        width: 205px;
        height: 80px;
        display: flex;
        justify-content: center;
        align-items: center;

        .qube {
            position: relative;
            height: 60px;
            width: 185px;

            .front, .back {
                .full;
                transform: rotateX(-90deg);
                transition: transform 0.3s ease;
                position: absolute;
                color: #fff;
                line-height: 60px;
                text-align: center;
            }

            .front {
                background-color: #191919;

                &::before {
                    .centered;
                    .full;
                    content: '';
                    display: block;
                    z-index: -1;
                }

                &::after {
                    background: #fff;
                    content: '';
                    height: 155px;
                    left: -75px;
                    opacity: .4;
                    position: absolute;
                    top: -50px;
                    transform: rotate(35deg);
                    transition: all 0.4s cubic-bezier(0.19, 1, 0.22, 1);
                    width: 60px - 10px;
                    z-index: 1;
                }
            }

            .back {
                background-color: #d6d35d;
                color: #191919;
                transform: rotateX(0deg) translateZ(60px);
            }

            &:hover {

                .front {
                    transform: rotateX(0deg) translateZ(30px);
                }

                .back {
                    transform: rotateX(90deg) translateZ(30px);
                }
            }
        }

        &:active {

            .front {
                border-radius: 0;
                background-color: transparent;
                color: #191919;

                &::before {
                    background-color: #d6d35d;
                    transition: all 0.3s cubic-bezier(0.19, 1, 0.22, 1);
                }

                &::after {
                    left: 120%;
                    transition: all 0.3s cubic-bezier(0.19, 1, 0.22, 1);
                }
            }
        }
    }
</style>

# Пузырики при нажатии

Каждый пузырик задаётся отдельным background-image: radial-gradient(...). При анимации добавляется класс с keyframe, в котором изменяется положение каждого пузырика через background-position.

Click me! 
<template>
    <a href="#" v-on:click="animateButton" class="bubbly-button">Click me!</a>
</template>

<style lang="less">

        border-radius: 4px;
        border: none;
        position: relative;
        transition: transform ease-in 0.1s, box-shadow ease-in 0.25s;
        box-shadow: 0 2px 25px rgba(255, 0, 130, 0.5);

            text-decoration: none !important;
        }

        &:focus {
            outline: 0;
        }

        &:before, &:after {
            position: absolute;
            content: '';
            display: block;

<script lang="javascript">
    export default {
        methods: {
            animateButton(e) {
                e.preventDefault();
                e.target.classList.remove('animate');
                e.target.classList.add('animate');
                setTimeout(function () {
                    e.target.classList.remove('animate');
                }, 700);
            }
        }
    }
</script>
